Letterbox 0.24b3

Released September 07, 2009

Updated for Snow Leopard.

Details:

Updated to support Snow Leopard.
Not yet compatible with 64-bit Mail.
Added buttons to for feedback and donations.
Added keyboard shortcuts to change the preview pane position.
Changes to the preview pane position preference are immediately reflected in open windows.
Renamed "Normal" to "Thick" in preview pane divider preference.
Renamed "Layout" to "Preview Pane" in preferences.
Still incompatible with some Mail plugins that also use Sparkle for auto-updating.
Changing the preview pane position with the menu item or keyboard shortcut does not visibly change the value in the preference pane.
