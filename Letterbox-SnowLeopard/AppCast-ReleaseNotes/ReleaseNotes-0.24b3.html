Letterbox 0.24b3

Released September 13, 2009

Updated for 10.6.1, bug fixes, and progress towards 64-bit.

Details:

Major under-the-hood changes towards compatibility with 64-bit Mail in Snow Leopard.
Added compatibility declarations for 10.6.1 and 10.6.2.
Hopefully resolved hang-on-close bug. Probably still some memory leak left.
Still incompatible with some Mail plugins that also use Sparkle for auto-updating.
Changing the preview pane position with the menu item or keyboard shortcut does not visibly change the value in the preference pane.
